# Portfolio 2.0

This is a minimal Vite + React project pre-configured with Tailwind CSS, Lucide (icons) and Framer Motion.

How to use

1. Install dependencies:

   npm install

2. Start dev server:

   npm run dev

3. Build for production:

   npm run build

Files created

- `package.json` - dependencies and scripts
- `index.html` - Vite entry
- `src/main.jsx` - app entry
- `src/App.jsx` - example app using Tailwind, Lucide and Framer Motion
- `src/index.css` - Tailwind entry
- `tailwind.config.cjs` - Tailwind config
- `postcss.config.cjs` - PostCSS config

Notes

- This repo uses Vite as the dev server.
- Edit `src/App.jsx` to start customizing.
